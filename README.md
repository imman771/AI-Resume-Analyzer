# 🤖 AI Resume Analyzer

An AI-powered Resume Analyzer that evaluates resumes based on ATS (Applicant Tracking System) standards and generates a professional PDF report.

---

## ✨ Features

- 📄 Upload Resume (PDF)
- 📊 ATS Score Calculation
- ✅ Resume Completeness Analysis
- 🎯 Skill Matching
- ❌ Missing Skills Detection
- 💡 Smart Suggestions
- 📈 Skill Match Pie Chart
- 📑 Professional PDF Report
- 🗂 MongoDB Storage

---

## 🛠 Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- MongoDB
- PyMuPDF
- ReportLab
- Matplotlib
